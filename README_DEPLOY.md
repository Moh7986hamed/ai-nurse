# تعليمات تشغيل ونشر التطبيق

هذا التطبيق يعمل بشكل مستقل تماماً ولا يحتاج إلى إعدادات خارجية معقدة أو قاعدة بيانات سحابية (Firebase).

## خطوات التشغيل محلياً:
1. قم بتثبيت الاعتماديات:
   ```bash
   npm install
   ```
2. ابدأ خادم التطوير:
   ```bash
   npm run dev
   ```

## إعداد مفتاح الذكاء الاصطناعي (Gemini API):
لكي يعمل المساعد الذكي، يجب وضع مفتاح الـ API الخاص بك في ملف `src/services/geminiService.ts` في المتغير `apiKey`.
*(يمكنك الحصول على المفتاح مجاناً من Google AI Studio)*

## خطوات البناء (Build):
لتحويل الكود إلى ملفات جاهزة للنشر على أي استضافة (مثل Netlify أو Vercel أو GitHub Pages):
1. قم بتشغيل أمر البناء:
   ```bash
   npm run build
   ```
2. سيتم توليد الملفات النهائية في مجلد `dist`.

## ملاحظات هامة:
- تم إزالة جميع الارتباطات بـ Firebase تماماً.
- يتم تخزين المحادثات والبيانات محلياً في المتصفح الخاص بك (LocalStorage).
- التطبيق مصمم ليعمل كـ Single Page Application (SPA).
